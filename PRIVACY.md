I. Introduction

Thank you for using Xueyue Telegram Drive Bot (hereinafter referred to as "this application").

This application was created by individual developer Xueyue to enable secure file transfer and synchronization between Telegram and Google Drive.

This Privacy Policy explains how this application collects, uses, stores, and protects your data.

II. Data We Collect

This application does not actively collect any personally identifiable information from you.

During use, this application will only access the following data:

Google Drive File Access Permissions

Scope: https://www.googleapis.com/auth/drive.file

Purpose: To upload files you actively send to your Google Drive account.

Restriction: This application can only access and manage files you upload or create through this bot; it cannot access other files in your Google Drive.

Telegram Username and Message Content

Purpose: To identify the source of instructions and uploaded files.

Restriction: Only used for bot instruction responses; it will not be saved or shared.

III. Data Use and Storage

The authorization information (token.json) is stored only on the server running this bot or in your Render persistent storage volume /data directory to maintain the Google Drive authorization status.

This application will not upload your authorization credentials, files, or any data to third-party servers.

Telegram message content is not stored permanently on the server.

IV. Data Sharing

This application will not share, sell, or publicly disclose your data with any third party.

All authorization operations provided by the Google API are protected by the Google Privacy Policy.

V. Data Security

The developer has taken the following measures to ensure data security:

All access is encrypted via HTTPS.

Credential files have strictly limited permissions, allowing only process read and write access.

Servers are regularly updated to fix security vulnerabilities.
